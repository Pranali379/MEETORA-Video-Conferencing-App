## вҡҷпёҸ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Pranali379/MEETORA-Video-Conferencing-App.git
```

### 2. Move into the project

```bash
cd MEETORA-Video-Conferencing-App
```

---

## Backend Setup

Navigate to the backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Start the backend server:

```bash
npm start
```

---

## Frontend Setup

Open another terminal.

Navigate to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the React application:

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```
